Actividad 1 - Portafolio de Ejercicios del Tema 2
Materia: Programación Web

Este repositorio contiene la primera actividad práctica del Tema 2 de la materia de Programación Web. En este proyecto nos introducimos al desarrollo web moderno estructurando con HTML5, dando diseño con CSS3 y agregando interactividad con JavaScript.

El repositorio sirve como evidencia de trabajo y está publicado mediante GitHub Pages, utilizando un menú principal (`index.html`) en la raíz para navegar entre cada una de las prácticas.

Resumen de Aprendizaje

Ejercicios del 1 al 7: Fundamentos HTML5
En este bloque sentamos las bases de la web. Estructuramos documentos desde cero usando etiquetas fundamentales, encabezados, párrafos y formatos de texto. Aprendimos a implementar navegación con hipervínculos (internos y externos), insertar imágenes locales manejando rutas relativas, y a organizar el contenido de forma semántica (`<header>`, `<section>`, `<footer>`). También practicamos el acomodo de información usando listas ordenadas, desordenadas y anidadas.

Ejercicios 8 y 9: Diseño y CSS3
Dimos el salto hacia la presentación visual, separando la estructura del diseño con hojas de estilo externas. Construimos desde una tabla compleja para un horario escolar (usando `rowspan` y `colspan`) hasta un formulario de contacto profesional. Aquí aplicamos variables CSS y tipografías de Google Fonts.

Ejercicio 10: Interactividad con JavaScript
Nos introducimos a la lógica de programación frontend uniendo HTML, CSS y JS en archivos distintos. Desarrollamos una calculadora que convierte grados Celsius a Fahrenheit
